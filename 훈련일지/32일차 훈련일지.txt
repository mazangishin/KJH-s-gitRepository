1. 년도 % 4 == 0 윤년 
2. (1번의 조건을 만족하고) 년도 % 100 == 0 평년
2-1. (1번의 조건을 만족하고) 년도 % 100 != 0 윤년
3. (1번과 2번을 만족하고) 년도 % 400 == 0 윤년
